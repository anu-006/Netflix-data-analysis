Netflix Data Analysis 

This project explores a dataset of Netflix movies to uncover insights about genres, popularity, ratings, and release trends. Using Python libraries like "pandas", "NumPy", "matplotlib", and "seaborn", the notebook performs data cleaning, transformation, and visualization to answer key business questions.


Key Questions Answered-
1. What is the most frequent genre of movies released on Netflix? 
2. Which genres have the highest vote averages? 
3. Which movie has the highest and lowest popularity, and what are their genres? 
4. Which year had the most filmed movies? 
Dataset The analysis uses an offline dataset (`mymoviedb.csv`).
